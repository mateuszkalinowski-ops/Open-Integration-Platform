# SellAsist Courier Integration — v1.0.0

## Overview
SellAsist REST API integration for retrieving shipment data and shipping labels. This is a read-only integration — SellAsist manages shipment creation externally; this integrator retrieves labels and shipment information only.

## External API
- **Protocol**: REST (JSON)
- **API URL**: https://{account}.sellasist.pl/api/v1
- **Documentation**: https://api.sellasist.pl/
- **API Version**: 1.90.7

## Authentication
- `apiKey` — API key passed as a request header
- Account-specific subdomain in the API URL

## Key Features
- Shipment listing with filtering
- Individual shipment detail retrieval with label file
- PDF label retrieval and merging for batch printing
- Read-only integration (no shipment creation)

## Limitations
- **No shipment creation** — SellAsist does not expose shipment creation via API; shipments are created through the SellAsist platform UI or other integrations
- This integrator is label-retrieval focused: fetch labels generated by SellAsist for printing and WMS association

## Configuration
See `.env.example` in the integrator directory for required environment variables:
- `SELLASIST_API_URL` — API base URL (includes account subdomain)
- `SELLASIST_API_KEY` — API key
- `SELLASIST_ACCOUNT` — Account subdomain name

## Sandbox
No dedicated sandbox environment. Use a test SellAsist account for development.

## Rate Limits
No documented rate limits. Follow standard retry with exponential backoff.
