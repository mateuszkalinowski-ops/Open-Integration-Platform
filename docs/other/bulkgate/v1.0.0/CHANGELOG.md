# Changelog

## [1.0.0] - 2026-02-24
### Added
- Initial BulkGate SMS Gateway connector implementation
- Transactional SMS via Simple API v1.0
- Promotional/Bulk SMS via Simple API v1.0
- Advanced transactional SMS via Advanced API v2.0 with template variables and multi-channel cascade (SMS, Viber)
- Credit balance check via Advanced API v2.0
- Delivery report webhook endpoint
- Incoming SMS webhook endpoint
- Full sender ID type support (system, short code, text, own, profile)
- Unicode SMS support
- Scheduled sending (unix timestamp and ISO 8601)
- Duplicate message prevention
- Comprehensive test suite (schemas, API client, FastAPI endpoints)
- Docker and docker-compose configuration
