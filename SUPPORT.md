## Getting Help

### GitHub Discussions

For questions, ideas, and community support, use [GitHub Discussions](https://github.com/pinquark/open-integration-platform/discussions):

- **Q&A** — Ask questions about setup, configuration, or usage
- **Ideas** — Propose new features or connectors
- **Show and Tell** — Share your integration workflows and setups

### Bug Reports

Found a bug? [Open an issue](https://github.com/pinquark/open-integration-platform/issues/new?template=bug_report.yml) using the bug report template.

### Feature Requests

Want a new connector or feature? [Open an issue](https://github.com/pinquark/open-integration-platform/issues/new?template=feature_request.yml) using the feature request template.

### Security Vulnerabilities

**Do not** report security vulnerabilities via public issues. See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.
