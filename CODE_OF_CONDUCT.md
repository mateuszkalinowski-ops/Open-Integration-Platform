# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. All contributors, maintainers, and participants are expected to uphold these standards.

## Reporting

If you experience or witness unacceptable behavior, please report it by emailing **office@pinquark.com**.

All reports will be reviewed and handled confidentially.
